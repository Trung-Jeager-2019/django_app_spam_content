Develop a NLP Model in Python & Deploy It with Django

Considering a system using machine learning to detect spam SMS text messages. My ML systems workflow is like this: Train offline -> Make model available as a service -> Predict online.

A classifier is trained offline with spam and non-spam messages.

Naive Bayes classifiers are a popular statistical technique of e-mail filtering. They typically use bag of words features to identify spam e-mail. Therefore, I’ll build a simple message classifier using Naive Bayes theorem.

The trained model is deployed as a service to serve users.

Having prepared the code for classifying SMS messages in the previous section, we will develop a web application that consists of a simple web page with a form field that lets us enter a message. After submitting the message to the web application, it will render it on a new page which gives us a result of spam or not spam.
